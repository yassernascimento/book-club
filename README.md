# BookClub

## First Steps

- Install Node 14 LTS locally
- Install Postgres 13 locally
- Then run:
  > npm i

## Ng App

### Serve App

- Run:
  > npx nx serve book-club-app

## API

### First-Time Config

- Update the `.local.env` file
- Then run:
  > npm run db-create

### Running Tests

- Run:
  > npm run test:api

### Serve API

- Run:
  > npx nx serve api
